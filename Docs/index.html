Installation

This script requires the following system requirments to run:
  • Apache, Nginx or IIS Sever
  • MySQL 5.0+ or MariaDB 10+ (MySQL 8+ supported)
  • PHP 8.2+
  • PHP Extensions: mysqli, mbstring & curl & gd & fileinfo & zip
  • PHP Funcrions: mail()
  • PHP Runtime Configuration: allow_url_fopen = On
You can ask your hosting company support to make sure to install/enable them for you
Delus requires SSL certificate only if you will enable video/audio chat or live streaming or to build the mobile apps Check the FAQs to learn more
To run Delus on Nginx & IIS severs, Check the FAQs to learn more

How to Install

  1. Now make sure you created an empty MySQL database along with user account for it.
  2. After unzip Delus Files, you will found Script Folder
  3. Start your FTP client (Like Filezila which is free) and login. Login to your server. You will need your host name (often times this will be your domain name), username, and password that you setup with the hosting company.
  4. Locate the domain folder you wish to upload your Delus files to on your server. You are going to upload all files located in Script Folder to your FTP client.
  5. Once the files are done uploading, you have to change chmod of content/uploads folder and content/themes/default/templates_compiled folder to 777.
  6. Just goto your website and Delus installer wizard will run automatically https://yourdomain.com/path_to_script
  7. In case you got the following error "wizard cannot create the config file" just make sure that the includes folder is writable "777" then go and empty the database then try to run the installer wizard again and after installation change it back to its normal permissions.

  8. Delus installer wizard is very clean, you will need to put your Purchase Code at step #3 and the rest is simple just follow the video tutorial

Facebook

After installing Delus now you can in easy steps integrate Facebook with it to let your users login with their Facebook accounts

  1. First goto: https://developers.facebook.com and Sign in with your Facebook account

  2. Now click My Apps and click Create App

  3. Fill the form and click Create App ID

  4. Now go to Settings and enter your domain or subdomain where you installed Delus in the App Domains

  6. Now go to Use cases then Use cases then Settings and put the following URL as Valid OAuth redirect URIs then click Save Changes

    If you installed Delus in your domain directly like www.domain.com the URI should be like this:
    https://www.domain.com/connect/facebook

    If you installed Delus in your domain directly but without www like domain.com the URI should be like this:
    https://domain.com/connect/facebook

    https://sub.domain.com/connect/facebook

    https://www.domain.com/directory/connect/facebook

  6. Also make sure you added the Bucket Policy by adding this JSON code in Bucket Policy, You'll need to replace "YOUR-BUCKET-NAME" with your full bucket name 

    {
  "Version": "2012-10-17",
  "Statement": [
    {
            "Sid": "AllowPublicRead",
            "Effect": "Allow",
            "Principal": {
                "AWS": "*"
            },
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::YOUR-BUCKET-NAME/*"
        }
    ]
}

  7. Also make sure you configered the CORS configuration by adding this JSON code 

    [
    {

            "POST",
            "DELETE"

        "AllowedOrigins": [
            "*"
