==========================================================
== If you running Delus v4.0.6 you can upgrade to v4.1 ==
==========================================================

!!! IMPORTANT !!!
==============================================================================================================
== Backup your database and your website files just in case something wrong happened                        ==
== Uninstall any Delus plugin(s) from your Delus both files and any database changes made by this plugin  ==
== If you can't do it ask the plugin author(s) for support and help                                         ==
==============================================================================================================

You can check our video tutorial from here: https://www.youtube.com/watch?v=Jch1GaWuBbE

Note: Make sure you are running on the default Delus theme 

1- Shutdown your system from the admin panel

2- Upload all files located in the "Script" folder and replace all files and folders, Except:

-- content/backups (folder)
-- content/uploads (folder)
-- includes/config.php (file)

3- Copy "update.php" file from "Update 4.0.6 to 4.1" folder to your server

4- Run the "update.php" file (http://yourdomain.com/path_to_script/update.php)

5- Empty this folder "content/themes/default/templates_compiled"

6- Copy The followings folder to your uploads folder "content/uploads"

-- Script/content/uploads/merits
-- Script/content/uploads/pwa
-- Script/content/uploads/temp (Give it 0777) Permission

=========================================================
== After Update ==
=========================================================

1- Remove any browser cache and cookies ( Check this to learn how: https://bit.ly/DelusClearCache )

2- Only for Cloudflare users, Don't forget to flush your website cache from Cloudflare

3- Go to your admin panel (http://yourdomain.com/path_to_script/admincp) and make site live again

4- Delete "update.php" from your script folder after updating

And That's it, Have Fun :)
